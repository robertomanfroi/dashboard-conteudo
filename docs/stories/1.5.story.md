# Story 1.5 — Rastreador de Concorrentes

**Status:** Draft
**Epic:** Dashboard de Conteúdo
**Agente:** @dev
**Depende de:** Story 1.4 (concluída)

---

## Descrição

Módulo para adicionar e monitorar perfis de concorrentes em múltiplas redes sociais com tabela ordenável.

---

## Prompt para Claude Code

```
Crie um dashboard de rastreamento de concorrentes.

Ele deve permitir adicionar perfis, usuários ou canais de
concorrentes e exibir:
- posts recentes
- engajamento
- frequência de postagem
- tendências de crescimento

Deve acompanhar esses concorrentes em múltiplas redes sociais.

Puxe dados publicamente disponíveis e apresente tudo em uma
tabela ordenável com tema escuro.
```

---

## Critérios de Aceite

- [ ] Formulário para adicionar concorrente (nome + rede social + handle)
- [ ] Tabela ordenável com: posts recentes, engajamento, frequência, crescimento
- [ ] Suporte a múltiplas redes: Instagram, YouTube, TikTok
- [ ] Indicadores visuais de tendência (↑ ↓ →)
- [ ] Dados fictícios de 5+ concorrentes populados

---

## Tasks

- [ ] Criar formulário de adição de concorrente

- [ ] Criar indicadores de tendência
- [ ] Criar mini gráficos de crescimento por linha
- [ ] Popular com dados fictícios realistas
