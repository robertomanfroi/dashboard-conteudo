# Story 1.6 — Consolidador de Notícias

**Status:** Draft
**Epic:** Dashboard de Conteúdo
**Agente:** @dev
**Depende de:** Story 1.5 (concluída)

---

## Descrição

Agregador de feeds RSS com cards de notícias, filtros por tópico e visual clean em tema escuro.

---

## Prompt para Claude Code

```
Crie um consolidador de notícias que agregue as últimas notícias
sobre [SEU NICHO] a partir de feeds RSS.

Exiba:
- manchete
- fonte
- data de publicação
- pequeno resumo

Adicione filtros por tópico, como:
- ferramentas
- pesquisa
- negócios

Use uma interface limpa, com cards em tema escuro.
```

> Substituir [SEU NICHO] pelo nicho do projeto antes de enviar.

---

## Critérios de Aceite

- [ ] Busca por palavra-chave
