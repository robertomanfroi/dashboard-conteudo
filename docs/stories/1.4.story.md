# Story 1.4 — Calendário de Conteúdo

**Status:** Draft
**Epic:** Dashboard de Conteúdo
**Agente:** @dev
**Depende de:** Story 1.3 (concluída)

---

## Descrição

Visualização mensal de posts agendados e publicados com chips coloridos por plataforma e filtros.

---

## Prompt para Claude Code

```
Crie uma página de calendário de conteúdo que exiba posts
agendados e posts já publicados em uma visualização mensal.

Cada dia deve poder conter múltiplos conteúdos exibidos como
chips coloridos.

Inclua filtros por plataforma, como Instagram, YouTube e outras.

Use uma interface escura, limpa e organizada.
```

---

## Critérios de Aceite

- [ ] Calendário mensal com navegação de meses
- [ ] Múltiplos chips por dia (cores por plataforma)
- [ ] Filtros por plataforma: Instagram, YouTube, TikTok
- [ ] Clique no dia abre detalhes do post
- [ ] Visual escuro, limpo e organizado
