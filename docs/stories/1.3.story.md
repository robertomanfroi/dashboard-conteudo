# Story 1.3 — Dashboard de Analytics

**Status:** Draft
**Epic:** Dashboard de Conteúdo
**Agente:** @dev
**Depende de:** Story 1.2 (concluída)

---

## Descrição

Painel de métricas com KPIs, gráficos de barras e linha, seletor de datas e ranking de melhores posts.

---

## Prompt para Claude Code

```
Crie uma página de analytics com gráficos de barras e gráficos
de linha mostrando métricas de performance de conteúdo das minhas
redes sociais.

Use o Metricool como fonte de dados.

Inclua:
- total de impressões
- taxa de engajamento
- crescimento de seguidores
- posts com melhor desempenho

Use tema escuro e inclua um seletor de datas.
```

---

## Critérios de Aceite

- [ ] KPIs em cards no topo: impressões, engajamento, seguidores
- [ ] Gráfico de barras (volume de posts/impressões)
- [ ] Gráfico de linha (crescimento de seguidores)
- [ ] Seletor de período (7d / 30d / 90d)
- [ ] Tabela com top posts por desempenho
- [ ] Dados fictícios realistas populados
