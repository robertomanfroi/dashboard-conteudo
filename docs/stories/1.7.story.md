# Story 1.7 — Refinamento Visual Premium

**Status:** Draft
**Epic:** Dashboard de Conteúdo
**Agente:** @dev
**Depende de:** Story 1.6 (concluída)

---

## Descrição

Refinar todo o dashboard para nível visual premium — aparência de produto SaaS profissional — e garantir consistência de UI em toda a aplicação.

---

## Prompt 1 — Visual Premium

```
Agora refine todo o dashboard para deixá-lo mais premium,
moderno e visualmente impressionante.

Quero um visual de software de alto nível, com:
- melhor hierarquia visual
- espaçamentos mais elegantes
- cards mais refinados
- melhor uso de contraste
- gráficos mais bonitos
- tabelas mais limpas
- aparência geral de produto SaaS premium

Mantenha o tema escuro e preserve a estrutura já criada.
```

---

## Prompt 2 — Consistência de UI

```
Revise todo o projeto e padronize:
- espaçamentos
- tamanhos de fonte
- bordas e sombras
- cores e botões
- badges, tabelas e cards
- formulários

Quero consistência visual em toda a aplicação,
mantendo tema escuro e aparência moderna.
```

---

## Prompt 3 — Dados Fictícios

```
Adicione dados fictícios realistas em todas as páginas para que
o dashboard fique visualmente completo e pronto para apresentação.

Os dados devem parecer reais, coerentes e bem distribuídos
entre as seções.
```

---

## Critérios de Aceite

- [ ] Visual consistente em todas as 6 páginas
- [ ] Hierarquia tipográfica clara e elegante
- [ ] Espaçamentos padronizados (design tokens)
- [ ] Cards, tabelas e formulários com mesmo estilo
- [ ] Dados fictícios em todos os módulos
- [ ] Aparência de produto SaaS premium
