# Story 1.2 — Gestor de Instagram

**Status:** Draft
**Epic:** Dashboard de Conteúdo
**Agente:** @dev
**Depende de:** Story 1.1 (concluída)

---

## Descrição

Módulo de gestão de posts do Instagram com visualização de status, formulário para adicionar ideias e layout em cards.

---

## Prompt para Claude Code

```
Crie um dashboard de gestão de conteúdo para Instagram.

Ele deve exibir:
- posts agendados
- rascunhos
- conteúdos publicados
- backlog de ideias

Inclua a possibilidade de adicionar novas ideias de posts com:
- legenda
- tipo de post
- status
- data de agendamento

Use uma interface escura, limpa e moderna, com layout baseado em cards.
```

---

## Critérios de Aceite

- [ ] Seção com 4 colunas/abas: Agendados / Rascunhos / Publicados / Backlog
- [ ] Cards com: legenda, tipo de post, status e data
- [ ] Formulário para adicionar novo post
- [ ] Filtro por status funcional
- [ ] Visual escuro, limpo e moderno

---

## Refinamentos Sugeridos (pós-validação)

- Filtros por formato (Reel, Carrossel, Story)
